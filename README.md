## 吐槽
课程提供的**前端代码、设计文档质量差**（变量命名的不协调；表意不清；前端代码与入口文档不统一……），大作业需求还存在自相矛盾的情况。  
由于上述槽点的存在，**与其认为大作业是对 后端基础开发能力 的考核，不如看做对 修复前后端交互BUG能力 的考核**。
而这就对前端代码的认知有了一定的要求，**仅凭微专业本身的内容肯定是不够的**。
